# DoD (Definition of Done) = Критерии готовности — любая лабораторная

- [ ] База разворачивается «с нуля» по шагам из книги.
- [ ] Объекты находятся в нужных подсистемах; формы и команды доступны.
- [ ] Движения в регистрах корректны; отчёты/запросы дают правильный результат «на дату».
- [ ] Печатные формы (если требуются) подключены и печатаются.
- [ ] Обработаны ошибки/проверки данных; понятные сообщения пользователю.
- [ ] Репозиторий: осмысленные коммиты (`feat/fix/refactor`), без мусора.
- [ ] Краткий отчёт с шагами проверки и 2–5 скриншотами в Google Docs.
