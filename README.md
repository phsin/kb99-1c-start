# Курс обучения 1C Старт с нуля 

Цель: дать уверенную базу разработки на 1С:Предприятие 8.3, научить работать со справочниками, документами, регистрами, СКД и интеграциями. 
Итог: 
1. мини-решение «Мини-Склад» 
2. базовая интеграция с ФГИС «Меркурий».

## Предварительные требования
- 1С:Предприятие 8.3 (Конфигуратор, Тонкий клиент).
  - Сommunity-лицензия [Как получить лицензию разработчика](https://dzen.ru/video/watch/64267e59c583f65e1b9b80c7?share_to=link)
- [Git bash](https://git-scm.com/install/)
- [1C Управление Торговлей Демо](https://its.1c.ru/download/utovio11.5.11/)

## Основные источники
- [Практическое пособие разработчика 1С 8.3](https://its.1c.ru/db/pubdevguide83) (для получения доступа к книге обратитесь в [группу Телеграмм](https://t.me/+n4r-kQA_LzUwMmM6))
- [Интеграция «Меркурий» для 1С 8](https://docs.kb99.pro/s/vetis-for-1cv8): общая страница и разделы (установка, параметры, реквизиты API). 

## Как учимся
* Каждую неделю ставим цели, разбираем теорию и выполняем одну–две лабораторные работы с четкими критериями сдачи и коротким отчетом.
* В конце недели — тест по теории и защита практического задания.

## Результат
К концу 6-й недели у вас:
- Мини-решение «Мини-Склад» (справочники, документы, регистры, отчёты, печатные формы).
- Набор запросов и проведение по нескольким регистрам (включая FIFO).
- Базовые права/роли, регламентные задания, простой HTTP-обмен.
- Отдельно — демонстрация ключевых шагов настройки интеграции «Меркурий» в 1С 8 (по чек-листу).

## Контрольные точки:

После каждой недели - мини-тест  
После 3 недель - промежуточный проект  
В конце - финальный проект с защитой
